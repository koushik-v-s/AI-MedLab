## ✨ Project Overview

The **AI-based Medical Laboratory** project is designed to transform laboratory medicine by leveraging artificial intelligence and machine learning. By optimizing laboratory processes, the system reduces human error, cuts down operational costs, and ultimately enhances patient outcomes and user satisfaction.

## 💡 Benefits and Considerations

### 🟢 Pros
- 🎯 Enhances the **precision** and **speed** of process execution.

### 🔴 Challenges
- 📊 Potential difficulties with reliable **data quality** and availability.

## 🛠️ Installation & Setup

### Prerequisites
- [Node.js](https://nodejs.org/) & npm installed
- [MongoDB](https://www.mongodb.com/) deployed and operational

### 1️⃣ Clone the repository
```bash
git clone https://github.com/koushik-v-s/AI-MedLab.git
cd AI-MedLab
```

### 2️⃣ Install backend dependencies
```bash
cd backend
npm install
```

### 3️⃣ Configure the environment 
Create a `.env` file within the `backend` directory and configure the environment attributes:
```ini
MONGO_URI=your_mongodb_uri
PORT=5000
```

### 4️⃣ Install frontend dependencies & Run
```bash
# In a new terminal tab
cd frontend
npm install
npm start
```

### 5️⃣ Launch the backend
```bash
# Head to backend directory and start
cd backend
npm start 
# or use nodemon for development
```
